## Quick Reference

| Param | Value | Notes |
|-------|-------|-------|
| `--ar` | Per prompt below | Aspect ratio |
| `--s` | 250 | Stylize — higher for more artistic |
| `--q` | 2 | Quality — max for final assets |
| `--style raw` | On lifestyle shots | Less Midjourney "look", more photorealistic |
| `--no` | text, logo, watermark, words, letters | Global negative for all prompts |

## Batch Workflow

### Recommended generation order:
1. **P1 first pass** — Generate all 7 P1 prompts, 4 variations each (28 images)
2. **P1 selection** — Pick best variant per prompt (7 winners)
3. **P1 post-production** — Color-correct, composite phone/card screens
4. **P1 review** — Approve with stakeholders
5. **P2 first pass** — Same workflow (24 images → 6 winners)
6. **P3 fill** — Generate remaining 5

### Post-Production Checklist (every image):
- [ ] Color grade matches diBoaS palette (warm + teal accent)
- [ ] Phone screens replaced with actual app UI composite
- [ ] Card faces replaced with actual diBoaS card from Figma
- [ ] No accidental text, logos, or brand marks visible
- [ ] Exported at correct dimensions for section
- [ ] Named per convention: `diboas-{section}-{variant}-v{n}.{ext}`
- [ ] Reviewed at both desktop and mobile crop

### Midjourney Tips for These Prompts:
- Use `--style raw` on all lifestyle/photography prompts for realism
- Skip `--style raw` on the illustration prompt (#15) and product shots
- Run `/settings` and ensure you're on v6.1+ for best photorealism
- For the split composition (#9), you may need to generate left/right halves separately and composite
- For phone screens: don't expect accurate UI — generate the scene, then composite real UI in Figma/Photoshop
- Upscale winners with `U1`-`U4` before downloading
